API: Shopify REST Admin API (no exemption required)
════════════════════════════════════════════════════

GET  /admin/api/2025-04/themes.json
     → List all themes (needs read_themes)

GET  /admin/api/2025-04/themes/{id}/assets.json
     → List all assets in a theme

GET  /admin/api/2025-04/themes/{id}/assets.json?asset[key]=...
     → Get single asset content (for download)

DEL  /admin/api/2025-04/themes/{id}.json
     → Delete a theme (needs write_themes)

Scope: write_themes (includes read_themes)

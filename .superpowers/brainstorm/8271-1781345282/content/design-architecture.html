API: Shopify GraphQL Admin API
├── themes(roles: []) → list all themes
├── theme(id:).files(filenames:) → get file contents
├── themeDelete(id:) → delete a theme
└── Scope needed: write_themes

Key Design Decisions

API Choice

GraphQL Admin API (required for new apps since April 2025). Queries: themes, theme.files. Mutations: themeDelete.

Download Strategy

Fetch all theme files via theme.files, bundle into a .zip on the server using JSZip, stream to browser.

Admin Auth

Secret key from ADMIN_SECRET_KEY env var, passed as ?key= query param. Validated in a middleware loader.

Cross-Shop Access

Admin reads all offline sessions from Prisma DB. Uses each shop's stored access token to make API calls on their behalf.
